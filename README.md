# VIGILA-v1.0-Biometric-Liveness-Detection-System
VIGILA es un nuevo estándar de seguridad digital que reemplaza los CAPTCHAs obsoletos por reconocimiento de vida en tiempo real. Utilizando visión artificial (OpenCV) y algoritmos de análisis de parpadeo, VIGILA asegura que detrás de la pantalla hay un humano real y no una IA o una fotografía.
